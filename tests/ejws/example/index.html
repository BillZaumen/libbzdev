Advantages

Just for starters,

  • The webnail can be placed behind a firewall or on a password-protected web server when confidentiality is important. For example, pictures of a product development team could be useful to a competitor if the members of the team can be identified. If you can find data on their past experience, that might suggest something about the product they are working on.
  • The web pages are useable with Javascript turned off - it is still possible to see all the webnail, although some navigation facilities will be missing, and a "slideshow" mode will not be available
  • When webnail are changed, the document object model can be modified easily. This provides considerable flexibility.
  • New page layouts can be added.
  • Multiple image formats can be used (e.g., PNG for some webnail).

Outstanding Issues

  • The javascript works properly with Mozilla and Safari, but not with Internet Explorer - issues with Internet Explorer, related to it not following certain W3C standards, have not be fixed at this point.
